# 前言

出一个补全计划的想法已然很久，一些想完成的事或者一些补全计划之前虽有在本本上写过几次，但总是寥寥几件，或许有些实现了，但大多最终都不了了之。可能是最近真正闲下来了或者是最近一年的经历让我产生了些许感想亦或许是觉得已然到了学生生涯的末期，不论哪种，总之现在想以博客的形式系统完整地记录保存一些我想在学生时代或者说是人生中想要做的一些事、看的一些番、读的一些书、玩的一些游戏种种。算是给自己二十几岁的岁月中留下一些痕迹吧。所选的书、番、游戏等都是我觉得还算优质的，倘若还有闲暇时间，我也会尝试写一些评测或随笔记录在博客中。九月八日凌晨小记。

# 正文

果然不出自己所料，等到想起继续写这篇小记的时候已经过了近二十天了。回想这二十天，不仅感叹又虚度了大半的时光。代码没写多少，基础知识没看，数学没咋学，没有规划目标，仿佛又回到了大三的颓废时光。每晚立志早起总会被手机拖跨，学习又被种种事情搞得心神不宁，二十天像是在梦里度过一般。唔，是要改变了，拖延症也要戒掉。如此三年下去，最终必然像去年一样在焦虑与悔恨中度过，但不一定会有和去年一样好的结果。

今年上半年经历了很多，可以算说是一个人生的转折点吧。上半年考研出分带给我惊喜，分数不仅过线还挺高了很多，原本以为考不上的我又有了学上，接下来准备复试、找导师、毕设、毕业、旅游让我原本单调的校园生活变得充实起来。毕业后回家享受假期的同时还穿插着 4 周暑期培训也算是让我在假期学了一点东西。培训结束又急急忙忙地考了科三科四拿了本，随后回到学校搬宿舍、买了自己的小电驴、自己动手买硬件装了台式，新生报到后便开始了 NWPU 研究生生活。开学那段时间被安排了写本子，最近在做图像拼接（现在做完了），之后马上就要开始上课了（已经开始上课了）。罗列下来，感觉自己还是有点忙的，但纵观四年大学下来摆烂的时间也不少，因此想给摆烂时候的自己找点活干，于是就有了以下补全计划。（虽然也是没学习，但总比一直玩一些辣鸡网游好）

# 游戏补全计划

最近别的没干，游戏倒是玩不少，但还是要适可而止了，现在已经过了网瘾少年的年龄，马上要和生活去对线，要少玩一点了，哎。

  1. 塞尔达 1-2 部
    第一部旷野之息还没玩完就开始第二部了，根本舍不得玩完 QAQ
  2. 双人成行
    和朋友一起，目前九月二十七日，现在快通关了，非常推荐和朋友一起玩
  3. 逆转裁判
    九月初完了一章，现在第二章还没结束，经典买了就不玩了
  4. 三伏
    还没买，有时间再说
  5. 只狼
    国区买了一份，只能说，打铁的声音真的很爽，真的很有成就感，用一句话说就是：不是角色变强了，而是现实的你变强了
  6. 空洞骑士
    拖了很久了，一直没继续玩

# 总结

本篇的小记只能说写了一半，这个系列也将一直写下去。书单仅写了两本，别的书也不知道看啥，只能先去图书馆转转，之后再补全，大概率是推理小说相关，我对此很感兴趣。除这些外，以后估计还会出一个专业相关的书单，包括一些基础课程的书籍等等，需要提高一下自己，为两年后找工作准备吧。还有一个项目的清单，准备学着写几个项目放到 GitHub 上，找实习和工作用，希望能坚持下去。
