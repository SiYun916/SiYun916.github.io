# 形式主义

突然写这篇博客，本意并不是想批判和吐槽最近遇到的一些事，而是想借此反思。这篇博客虽名为形式主义，我想说的却并非只有这一点。不知取何名，假以形式主义，更何况这是不是形式主义的另一种形式呢。

# 善恶

每个人从小到大都被教育惩恶扬善，一定要发扬各种美德诸如种种，我自不例外。所以以前的善恶观是黑白的，好人是要打到坏人的。随着年龄阅历的增长，伴同着互联网野蛮的发展，我感觉我的善恶观也在变，变成了灰的。至于人，我觉得可能会存在绝对的黑，不存在绝对的白。

看来，我是很支持荀子的性恶论的，或者说是我支持人生来是混沌的。最熟悉自己的只有自己，最关心自己的也只有自己，在巨大利益面前考虑的也只是自己，人天生应该是自私的、虚伪的。是后天的道德标准、信仰、精神指引我们要成为一个 “白” 的人，但在实际的社会生活中，迫于身份与压力，我们只能成为一个 “灰” 的人。

这样想来，那些人造的偶像又有什么资格成为偶像呢？既然我们向往的是一个不完美的人身上的部分优点，那为何我们不构造一个完美的神并为之钦慕呢？

# 形式

形式没错，我承认需要有形式的，并且它也很重要。但我觉得，相比形式内容更重要，形式主义有错。

我不知道有些东西是不是形式主义，但我真的很反感，但出于无奈又不得不做，甚至要好好做。真的备受折磨。

甚至不只是在学习工作中有，几乎到处都是。日常分享生活，我觉得不算。但绞尽脑汁把普通的东西包装精美，甚至不是自己的也要去蹭来用，并借此以或激动或平常的语气隐隐向人炫耀，这何尝又不是一种形式主义呢？

这种东西，真的是无处不在，无时不有。我可以不听不闻不问，但被强制安排的，如果有上帝就祈求你让我少遇到点吧。

# 因果

我不信佛，但我相信因果。因在前，但大多情况下只有看到了果才能串起因。所以啊，但知行好事。
