# 苍鹭与少年

清明假期的第二天，下午四点左右到了西市商场三楼。匆匆买了 60 个币，抓完娃娃（抓到了三个）便去了影院。在西安的话，并不推荐来这个大唐国际影城看电影，因为票价一般都比其他的要高十几块钱，而我喜欢在这的原因是工大新老校区坐车来回方便。这里吐槽一下，新老校区通勤的时间都够我回趟家了。

从去年开始，就不断听说 “你想活出怎样的人生” 大名，可惜国内久久没有引入上映。

在我很早的时候，可能是小学，就看了宫崎骏老爷子的千与千寻。但那时并不知道宫崎骏是谁，也不知道看的动漫是什么，记忆最深刻的除了主角就是无脸男。再后来，知道了宫崎骏的大名，但苦于懒和贪玩，并没有看过几部作品。刚好趁这次放假上映，去影院看一看，顺便给之前看的盗版电影补个票。

# 观影杂谈

剧情和隐喻在这里就不讲了，网上有更好的文章去讲解和分析，仅谈谈我的感受。这部片子，其实翻译成苍鹭与少年更好，而不是你想活出怎样的人生。后面这个给我的感受不大好，就好像是有人在刻意引导你该怎样去理解这部电影。

至于大道理什么的也就不讲了，讲多了会有一种浓浓的说教感。而且一千个人眼中有一千个哈姆雷特，不论这部电影在哪点上触动了你，只要对你有启示就行。

刚好最近在看史铁生的《我与地坛》，书中有句话很有哲理，大概意思是 “人活着不是为了寻找意义，而是寻找意义是为了活着。人就是欲望本身。” 在电影结尾，真人告别自己的母亲，打开原本世界的门勇敢走出去，欣然接受了新的生活。我想电影这里想传达给我们的信念大概就和书中一样吧。

还有？宫崎骏老爷子的动漫真的太有特点了，影片一开始就能感受到浓浓的 “宫崎骏画风”。哇啦哇啦也非常可爱！

# 思想摘录

在以后的年月里，还将有很多我料想不到的事发生，我仍旧有时候默念着 “上帝保佑” 而陷入茫然。但是有一天我认识了神，他有一个更为具体的名字 —— 精神。在科学的迷茫之处，在命运的混沌之点，人唯有乞灵于自己的精神。不管我们信仰什么，都是我们自己的精神的描述和引导。
