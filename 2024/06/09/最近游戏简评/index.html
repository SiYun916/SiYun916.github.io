# title: 最近游戏简评
date: 2024-05-29 20:44:58
excerpt: 上半年游戏玩的其实并不是很多，每周的游戏时间平均不到二十小时，最近开始组会汇报更是没有时间去玩。尤其晚上回宿舍太晚就不想玩了，一般看会直播就水饺了。有点能预见到工作后的生活了。
category: 游戏
tags: 游戏简评

# 24 年上半年游戏简评

# 艾尔登法环

如果你没有玩过这个游戏，那我极力极力推荐你去试试。让我印象最深不是 BOSS 战、不是故事，而是里面的风景、建筑与地图设计（墓地与恶心的沼泽除外）。地底星空、雪山、崩坏的法姆亚兹拉初见真的让人震撼。一周目玩的狗牙，等级碾压后打 boss 复读战绩有点轮椅。

# 苍之彼方四重奏

推了两年，本体终于推完了，四条线中还是最喜欢美咲线。最近还买了苍彼的周边和 switch 版的 extra 1&2。接下来先推美咲的 extra。黑长直大姐姐谁不喜欢捏。

# HADES

玩起来就停不下来，但如果满分是 5 分的话，这个游戏我打 4 分。对于我这种彩笔来说，真的太肝了，没看攻略，十几个小时通关一次。换个武器玩跟个新游戏一样，搞得我不想玩了，懒得去打真结局。

# 山河旅探

年初的一款本格推理游戏，水墨风加中国近代元素非常吸引我，希望快点出 DLC，也希望能够出更多这样的国产作品。

# 大侠立志传

这玩意是真的上瘾，可惜功法不能全学，一周目爽不到太难受了。

# 杂交版 PVZ

优秀的改版，玩起来直呼上瘾上瘾！

# 喜丧

愿天下不再有人口拐卖。不再有封建糟粕。

# 不同的冬天

虽然有点谜语人，但解密设计的不错。以及换位思考的重要性。

# Evan’s Remains

解密加叙事。需要耐心看下去。

# pubg

当老六的感觉真爽。

# 英雄联盟

我发现了低端局的真谛：只要自己方不投降，那么投降的一定是对面。低端局不仅玩的菜，嘴臭脾气大，还爱指挥。打信号不互动是最优解。

# 碧蓝幻想 relink

战斗！爽！不到 648 的一半你就可以爽至少 100 个小时。
