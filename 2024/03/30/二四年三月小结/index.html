# 三月小结

# 总结

又到月末，想再水一篇总结，把欠下的 blog 补全。

所以滑动鼠标滚轮，在 keep 上翻看归档的每日计划，仔细回想这个月干了哪些事。唔，这个月总算没太闲着了，也算有些许收获。让我分点列出：

  1. 游戏（玩）

    • 不同的冬天（和 ZYF 通关）：还是有点深度的一个游戏，但是剧情有点谜语人。

    • 碧蓝幻想 relink：演出真不错，主线就是爽游，但要玩得爽也要刷刷刷。奶刀真的帅！

    • 山河旅探：玩到第四章了，先不谈抄袭、借鉴什么的。实打实本格推理力作，并且结合了近代史家国元素，值得一玩。

    • 艾尔登法环：真的是属于，打开了不想关，关了不想打开。如果不是有学习任务，真的想玩一天。

    • Getting Over It：教研室折磨自己，锻炼心态的。

    • LOL：开黑

  2. 游戏（买）

    • 入手了 PSV

    • 碧蓝幻想 relink

    • 暗黑地牢：玩不来

    • 犹格索托的庭院

    • GRIS

    • SHADOW OF WAR

    • PORTAL

  3. 书（读）

    • 旅宿：已读完，值得二刷

    • 听听那冷雨：太专业的读不下去，就让我算是读完了吧

    • 我与地坛：没读完

  4. 项目相关

    • 实验室的交接代码看完了

    • 用 C++ 复现了论文 SPDMEF 的代码，并提交到了 github

    • 整理 pin 的项目，提交了第一个单词 "I"

    • C++、JAVA、Makefile、Boost 库、opencv、编程规范学习并整理到语雀

    • 2 天时间重装了电脑，自己装了机械硬盘，

  5. 杂项

    • 上课和考试

    • 乒乓球与网球

    • 师兄师姐拍毕业照吃饭

    • 逛校园和拍照

# 计划

书到现在应读九本，实际欠了五本半，估计今年计划泡汤了。blog 按计划算是水了上来。游戏买了也没玩多少（而且卡带还没买过），一个月玩了应该不到 50 个小时，少睡会觉吧。下个月真的得多读一点书了，还有主题开发要迅速提上日常。

哎，到现在没有工资，不然考虑买个 surface，提高一下效率，上课开发加上新增一块屏幕。

目前计划，下个月把 C++ 5th 那本书看完，以及 effictive c++ 看完，还有 json 那个项目。

实验室项目看看视觉相关的知识吧，还有 Qt 知识。

游戏，先把山河旅探和法环通关，然后是碧蓝。春物教研室玩吧，躺床上的时候玩一会苍彼。

论文，不想看，想找工作。

清明，出去玩一次，不算入旅游计划。
