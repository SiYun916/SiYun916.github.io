# 二月小结

# 小叙

一句话总结一下二月就是吃喝玩乐。

容我先用流水账的方式回忆一下我这个月都干了什么。2 月 3 日，坐车回石家庄，找发小玩了三天直到到腊月二十六晚才回家。之后的日子里不是在床上躺着玩手机，就是去网吧五排开黑，要么就是去串亲戚去吃饭，期间初中十几个同学还聚了一次。悠闲的时光过得总是很快，初九也就是 18 号回到学校，当晚得知有个作业没交后花了 30 分钟写完匆匆上交。之后两天就是拿快递和收拾宿舍，偶尔去教研室学一会。19 到 25 这周内，写了写两门课程的作业，IS 的代码改了一些，看了会书，其余全花费在游戏上。26 号用 latex 写完 Data Mining 这门课的报告，27 号看了 LT 的代码并改了改 IS 代码。

综上就是这一个月所干的全部事情，似乎没有什么特别值得记住的。

# FLAG 完成情况

  1. 游戏入手情况：《山河旅探》、《饿殍：明末千里行》及周边、《桃源村日志》及周边、《黑森町奇谭》
  2. 游戏通关情况：未通关任一游戏，二月在家主要玩 PUBG 和 LOL，回校后玩了苍彼美咲线、艾尔登法环、山河旅探。上述三个真的是好游戏，值得写评测。
  3. 读书情况：看完了阿加莎的《四魔头》（在回家的高铁上看的），只能说这本书不太推荐；还正在看余光中的《听听那冷雨》，有些文章值得细品，给我的整体感觉是辞藻华丽（中性），有些句子读不明白。相比于汪曾祺先生的散文来说，我还是更喜欢汪先生的文风，不过余光中先生的散文中许多段落都值得背诵摘抄。除此之外，复习了《项脊轩志》和《虞美人・听雨》，真的好喜欢这两篇啊，尤其是《项脊轩志》，越读感触越深。
  4. 入手了咖啡机，瑞幸的豆子，自己做了几次，味道只能说正统美式苦味，说不上好喝。需要学一学咖啡怎么做。
  5. Blog 主题没写，前端没学，JAVA 都没学。C++ 没怎么看（看了一点点中的一点点）。项目没进展，而且又多了一个活。
  6. 意外收获：回家换了一个手机苹果 13

# 后记

这个月没什么有意思的事情发生，月末这几天晚上都会 LOL 和同学开黑组排，争取上个翡翠。下个月唯一目标，早睡早起，写 Blog 主题。
